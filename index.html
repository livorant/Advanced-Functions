What are functions?

  1. A function is a rule that assigns a single, unique output to each input

  2. You can think of it as a machine where you put in an input, and it gives you one specific output

  3. The set of all possible inputs is the domain, and the set of all corresponding outputs is the range

  4. The relationship is often written as y = f(x), where x is the input and f(x) is the output
